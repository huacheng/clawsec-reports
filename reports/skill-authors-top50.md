> Last updated: 2026-03-22 | Data source: [ClawSearch Authors](https://clawsearch.cc/#authors) | Powered by [ClawSec](https://clawsec.cc)

## Overview

The AI agent skill ecosystem is growing rapidly. With **33,155 skills** indexed and **435+ deeply audited**, we compiled the first trust-based leaderboard of skill authors.

Each author is scored by ClawSec's Trust Score system, which evaluates:
- **Code Safety** — Static analysis and pattern matching results
- **Behavioral Safety** — Firecracker sandbox execution monitoring
- **Semantic Intent** — LLM-based analysis of what the code actually does
- **Track Record** — Historical audit results across all published skills
- **Community Signal** — Downloads, stars, and usage metrics

### Trust Levels

| Level | Score Range | Meaning |
|-------|-----------|---------|
| **Trusted** | 0.80+ | Consistently safe code, no malicious findings, strong track record |
| **Established** | 0.60–0.79 | Mostly safe, minor issues at most, growing trust |
| **Emerging** | 0.40–0.59 | Limited history or mixed results, use with caution |
| **Caution** | 0.20–0.39 | Suspicious patterns detected, review before use |
| **Dangerous** | <0.20 | Confirmed malicious activity in published skills |

## Key Insights

### 1. The Ecosystem is Young
Most authors are in the **Emerging** tier (score 0.40–0.59), reflecting the early stage of the AI agent skill ecosystem. Trust takes time to build.

### 2. Prolific ≠ Safe
Having many published skills doesn't guarantee safety. **wanng-ide** has 20 skills but 3 malicious ones. Always check individual skill audit reports.

### 3. Downloads Concentrate at the Top
**steipete** alone accounts for over **1.1 million downloads** — more than all other top-50 authors combined. The ecosystem follows a power-law distribution.

### 4. Zero-Download Authors Can Still Be Trustworthy
Several authors like **anmolnagpal** (30 skills, 28 safe) and **anugotta** (13 skills, 12 safe) have excellent safety records despite no download data, suggesting quality contributions that haven't gained visibility yet.

### 5. Malicious Authors Exist in Plain Sight
3 out of 50 top authors have malicious findings. These aren't fringe actors — they have thousands of downloads and mix malicious skills with legitimate ones.

## How to Use This Data

### Before Installing a Skill
```bash
# Check the skill's trust score
npx clawsearch-guard <skill-name>

# Search for safe alternatives
npx clawsearch "what you need"
```

### Check an Author's Track Record
Visit `https://clawsearch.cc/#author/<handle>` to see all their skills and trust metrics.

### Full Audit Reports
Visit `https://clawsec.cc/#skill/<skill-name>` for detailed 5-tier security analysis.

## Methodology

- **Data Source**: ClawSec audit database (33,155 skills indexed, 435+ deeply audited)
- **Audit Pipeline**: 5-tier analysis — static analysis, pattern matching, LLM semantic review, Firecracker sandbox execution, LLM final review
- **Trust Score**: Weighted composite of code safety, behavioral analysis, semantic intent, author track record, and community signals
- **Downloads**: Sourced from ClawHub registry API
